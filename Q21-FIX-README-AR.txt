تصحيح السؤال 21 من بنك DP-900 الكبير

الإجابة الصحيحة:
1) Image files -> Azure Blob storage
2) Key/value pairs -> Azure Table storage
3) Relationships between employees -> Azure Cosmos DB Gremlin API

ما تم إصلاحه:
- تحويل السؤال من تقييم ذاتي بصري إلى سؤال Matching تفاعلي يتم تصحيحه تلقائيًا.
- استبدال صورة الحل الخاطئة بصورة صحيحة.
- تصحيح الشرح وإضافة سبب واضح لكل اختيار.
- تجاهل الإجابة القديمة المحفوظة لهذا السؤال إذا كانت من نوع visual_review.

طريقة الاستخدام:
1) فك الضغط داخل فولدر المشروع الأساسي.
2) وافق على Replace files.
3) من CMD داخل المشروع:
   git add .
   git commit -m "Fix DP900 question 21 mapping"
   git push
4) انتظر نجاح GitHub Actions ثم افتح الموقع باستخدام Ctrl+F5.
